CQPDemo: Search the EUROPARL corpus with CQP query.

This is an experimental search interface for parallel corpora. It uses the Corpus Work Bench (CWB) as its backend and queries use the CQP query language.

Specify a query in the text field on the left-top corner of the upper frame and press "Run query" to start the search. Note that queries may take quite some time and due to the frames it will not be visible that the system is still busy. Don't repeat the request many times. Depending on the complexity of the query this can take several seconds to see any results.

You may also

  • select aligned languages to be displayed
  • increase the context to be displayed
  • look at the distribution of matches per document
  • display the frequencies of matching strings
You can also show more context for each match by clicking on "context" to the left of the resulting string.
