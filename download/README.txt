
OPUS - The Open Parallel Corpus
-------------------------------


This directory contains the downloadable files from OPUS, the open
collection of parallel corpora.  Tar-balls contain all corpus files
for a specific corpus and each sub-directory includes archives for
different parts of the corresponding parallel corpus. More information
about each corpus in the collection can be found in the README files
and on the OPUS-website: http://opus.nlpl.eu
