
OPUS - Europarl
---------------

Contact: jorg.tiedemann@helsinki.fi
Website: http://opus.nlpl.eu/Europarl.php
Source:  http://www.statmt.org/europarl/


Europarl is a parallel corpus extracted from the European Parliament
originally compiled by Philipp Koehn. The version on OPUS is based on
release 7.

xx.tar.gz ........... tokenized corpus files in language <xx> (XML format)
xx.raw.tar.gz ....... untokenized (raw) corpus files
xx.parsed.tar.gz .... UD-parsed corpus files
xx-yy.xml.gz ........ standoff sentence alignments files in XCES format
xx-yy.txt.zip ....... aligned plain text files (Moses format)
xx-yy.tmx.gz ........ bilingual TMX files

sub-directories:

mono ................ monolingual plain text files
freq ................ word frequencies
